How to find your Notion Database ID

The Database ID is part of the Notion URL when viewing the database as a page.

  1. Open the page that contains the database (table, board, list, etc.). Ideally, open the database in full-page view.
  2. Copy the page URL from your browser’s address bar.
  3. From the URL, copy the 32-character ID right after notion.so/ and before ?v=.
    https://www.notion.so/2578e582cda5807891fdedbdb7dd30da?v=2578e582cda5804bbd04000c1968a4db
    Paste the highlighted part into the “Notion Database ID” field on the home page.
    Tip: Some URLs may show hyphens (UUID form). You can paste either the 32-character form (no hyphens) or the hyphenated UUID; both are accepted by the API.

How to obtain your Notion API token

Follow these steps to create or use an Internal Integration in Notion and retrieve its secret token. Open Notion Integrations: notion.so/profile/integrations.

  1. Create a new integration or open an existing one with the right privileges, and ensure it is connected to the correct pages/databases.
  2. If creating a new integration: enter a name, choose your workspace, set the type to Internal, then click Save. Next, click Configure integration settings to adjust permissions.
  3. Select the page where you want to delete the rows in the Access tab.
  4. Copy the Internal Integration secret in the Configuration tab. Treat it like a password; you will paste it in the application’s home page when starting a deletion run.
Keep the token secret. Prefer using a dedicated integration scoped only to the target database. Rotate/revoke if exposed.
